🔹 Upcasting

👉 Upcasting is the process of converting a subclass object into a superclass reference automatically.

🔹 Downcasting

👉 Downcasting is the process of converting a superclass reference back to a subclass type explicitly.


| Feature   | Upcasting           | Downcasting       |
| --------- | ------------------- | ----------------- |
| Direction | Child → Parent      | Parent → Child    |
| Casting   | Automatic           | Manual            |
| Safety    | Safe                | Risky             |
| Access    | Parent methods only | Child methods too |
